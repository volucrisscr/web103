CODEcamp - Web 101

Course Overview

Why should I take Web Basics 101?

This entry level course teaches students essential skills in HTML (HyperText Markup Language) and CSS (Cascading Style Sheets). The techniques taught are those used by professional Web developers and provide students with a strong foundation to build their skills and create robust and scalable Web sites. Whether you want to develop a Web presence for your business or find a new career in Web design, this course will get you started.

What will I learn?

This course will teach you how to create multi-column Web page layouts that include components such as images, menus, lists and tables. Along the way, students will gain an understanding of how professional websites are developed and learn about the applications used to develop them.

What can I expect?

Each 2.5-hour session is a mixture of Lecture and hands-on Exercises to best teach you the fundamentals of Web design. Hands-on practice is essential to understand and fully grasp the concepts presented. The class is limited to 14 students.

Class 1 - Intro to Web Programming, Eclipse Installation and Github Pages

In this class you will receive an introduction to basic web programming using the 3 fundamental web technologies: HTML, CSS and JavaScript! You will learn the fundamental responsibility of each technology, also known as "Separation of Concerns" (Content vs. Presentation vs. Functionality). You will also see the basic layout of an HTML5 page, covering in detail all of the HTML elements that make up a very basic HTML layout. From here you will learn how to embed external JavaScript and CSS stylesheets into your page.

You will also receive step-by-step instructions on how to install your Eclipse Integrated Development Environment (IDE). You will use Eclipse as a tool for developing as well as learning how to program in HTML, CSS and JavaScript. You will install the Git and Web plugins, which will be used to program your web pages and integrate your work into Github.

After installing and setting up your Eclipse environment, you will set up your very own Github page! This Github page serves as your class project that you will have an opportunity to present at the end of the last day of this course. Your Github page is webpage that you will be able to program from scratch and will be able to leverage Github to host it.

Class 2 - The DOM and Adding Style to Your Github Page

This class introduces the concept of the Document Object Model, or DOM. The DOM is an important concept to understand when applying styles to specific elements on an HTML page.

You will also learn how CSS stylizes your HTML. You will learn the 3 methods for embedding styles in your HTML pages. This class covers the basics of CSS styles and selectors. You will use what you learn in this class to apply these styles to your very own Github page that you created the previous class. You will learn how you can leverage CSS selectors to manipulate very specific elements in your HTML page.
