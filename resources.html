General Use

  • JavaScript in one pic - A view of JavaScript and its scope

  • IT-eBooks - Completely free PDF-form textbooks for HTML, CSS, JavaScript, and many more skills!
  • Placehold.it - As seen in class, you can use this to generate images of any size.
  • Ajaxload - Ever see those little "spinners" when you're loading content? Those are just .gif images being shown while an AJAX request is being fired (AJAX = asynchronous JavaScript and XML). When the callback function completes successfully, the spinner goes away. This site let's you create really cool custom AJAX spinners that you can then save as .gif files for use in your websites
  • Media4X - Compress big-size JPEG/PNG files

  • Browserstack - Great tool for emulating various browsers
  • Tool for debugging the notorious IE
  • Deminify JavaScript or HTML
  • Indent/Tabify HTML/CSS/JavaScript
